== Description ==
Easy Math Captcha for Contact Form 7 is light weight plugin. it allows you to add math captcha to your form. this way you will get the spam protection.


= Main Features =
* Light weight plugin
* Easy to install and use
* Single digit easy captcha
* 100% multilingual

== Installation ==
Following the steps below, you can get the post views count form up and running in seconds.

1. Upload 'cf7-easy-math-captcha' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
